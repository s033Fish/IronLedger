# Iron Ledger (Expo / React Native)


A retro-inspired personal weightlifting log with offline storage, XP & leveling, and PR tracking. No accounts, no sync.


## Spec highlights
- Log sets (exercise / weight / reps)
- Local-only persistence via AsyncStorage
- Day-best PR graph per exercise (Epley 1RM)
- XP: +2 per set, +10 per PR
- Leveling: XP required = 100 * n^1.25 (cap 100)
- Retro iron aesthetic, Bebas Neue + Inter


## Run
```bash
npm i
npm run start
